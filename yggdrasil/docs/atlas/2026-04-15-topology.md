## What moves

| Moving | From | To | Phase |
|---|---|---|---|
| All transformer inference | Ollama on Munin + Hugin | vLLM via llama-swap on Hugin | F |
| all-minilm embeddings | Ollama | TEI `:11435` | F |
| rwkv-7-g1e memory classifier | Ollama | ai00-server `:11436` | F |
| `model` field in chat body | Required | Always omitted (Fergus intent-routes) | done (Sprint 068) |
| Cross-model KV reuse | none | KVCOMM anchor pool across every swarm pair | G |
| KV memory footprint | FP16 full | TurboQuant 3-bit K / 2-bit V + recent-128 FP16 | H |
| KV persistence across restarts | none | LMCache cold NVMe tier, dreamer prewarm pre-loads | I |
| Agentic flow streaming | buffered final StepDelta | per-tool-call SSE frames visible live | K |
| Secrets (gitea, ha_token, brave) | systemd env + /opt/yggdrasil/.env | Mimir vault via `{{secret:NAME}}` | L |
| 8 service VULN/FLAW gates | xfail | passing | C |
